Description of the STEP data for the open source project "Humanoid Open Platform", as of 1 October 2015

In 2013, igus® GmbH in Cologne started to design and develop plastic components as connecting elements for NIMBRO, 
a teen-size football robot created by the Institute of Computer Science VI (Professor Behnke) of the University of Bonn. 
This is being done in close collaboration with the institute and with the support of Studio Ambrozus, an industrial design company in Cologne. 
On the university side, the project is being supported by the Deutsche Forschungsgemeinschaft (DFG = German research association). 
igus® is not receiving any grants for this. The published STEP data represent the current status of development. 
These data can be used to make the outer shell of the robot in a 3D-printing procedure.
In addition to the 3D STEP data, the PDF of a 4-page flyer is enclosed for documentation purposes.
igus® is making this software available from October 2015 onwards as open source data under the Creative Commons 
licence (http://creativecommons.org/licenses/by-sa/3.0/legalcode).

Contents:
37 STEP files in one ZIP file. These show the individual parts of the entire shell of the robot's body.
1 PDF file: "humanoid open platform" (igus® GmbH, as at 2014)

Beschreibung der STEP Daten für das open source Project „humanoid open platform“ Stand 01.10.2015

Die igus® GmbH in Köln hat 2013 damit begonnen für den existierenden teen size Fußballroboter NIMBRO der Universität Bonn, 
Institut für Informatik VI (Professor Behnke) Kunststoffteile als Verbindungselemente zu entwerfen und entwickeln. 
Dieses geschieht in enger Kooperation mit dem Institut und mit Unterstützung der Kölner Industrie-Designfirma Studio Ambrozus. 
Das Projekt wird auf Seite der Universität durch die Deutsche Forschungsgemeinschaft (DFG) unterstützt. igus® erhält hierbei keinerlei Fördergelder. 
Die veröffentlichten STEP Daten stellen den aktuellen Stand der Entwicklung dar. 
Mit diesen Daten kann die Außenhülle des Roboters im 3D Druck Verfahren hergestellt werden.
Neben den 3D STEP Daten ist das PDF eines 4-seitigen flyers zur Dokumentation des Projektes enthalten.
igus® stellt diese Software ab Oktober 2015 als Open Source Daten unter der Creative commons 
Lizenz (http://creativecommons.org/licenses/by-sa/3.0/legalcode) zur Verfügung.

Inhalt:
37 STEP Dateien als ZIP. Diese stellen die Einzelteile der gesamten Körperhülle des Roboters dar.
1 PDF Datei: „humanoid open platform“ (igus® GmbH, Stand 2014)
